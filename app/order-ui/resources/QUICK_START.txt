📝 CHUẨN BỊ:
   1. Chuẩn bị file logo: icon.png (1024x1024 px, PNG, nền trong suốt)
   2. Chuẩn bị splash: splash.png (2732x2732 px, PNG)

🚀 THỰC HIỆN:
   
   Bước 1: Đặt file icon.png và splash.png vào thư mục này (resources/)
   
   Bước 2: Mở terminal tại thư mục order-ui và chạy:
   
      npm run generate-assets
   
   Bước 3: Build lại app:
   
      npm run build
      npx cap copy android
      npx cap open android

✅ XONG! Logo mới đã được cập nhật.

📖 Xem README.md để biết thêm chi tiết và cấu hình nâng cao.
